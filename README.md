### ğŸŒ TCP & UDP Protocol Analysis (Wireshark)
Independent network traffic analysis project examining **transport-layer behavior** using real packet captures.  
This project focuses on understanding how TCP and UDP differ in reliability, sequencing, and communication models.

**Key highlights:**
- Captured and analyzed FTP (TCP) and TFTP (UDP) traffic using Wireshark
- Identified TCP three-way handshake, sequencing, acknowledgments, and header fields
- Analyzed UDP packet structure, checksum behavior, and connectionless communication
- Compared reliability and performance trade-offs between TCP and UDP
- Troubleshot Linux permission issues related to root-owned packet captures
- Produced a formal technical report with diagrams, tables, and packet evidence
